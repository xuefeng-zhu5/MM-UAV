# A Tri-Modal Dataset and a Baseline System for Tracking Unmanned Aerial Vehicles

## [**Dataset Website**](https://xuefeng-zhu5.github.io/MM-UAV/)

<center><img width="90%" alt="" src="./figs/samples.jpg"/></center>

## Main Feature
* **_More modalities_**: RGB, Thermal Infrared, and Event
* **_Large-scale dataset_**: 1,321 tri-modal sequences, 2.8M annotated frames
* **_Comprehensive evaluation_**: Anti-UAV tracking, Multi-object tracking
* **_Generic scene and diverse object category_**: >30 scenes
